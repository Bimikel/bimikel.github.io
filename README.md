# Personal Site

Static personal website. Includes light/dark theme toggle, resume embed, activities/favorites sections, and contact links.

## Live Site

https://bimikel.github.io/

## Run Locally

1) Clone the repo  
`git clone https://github.com/Bimikel/bimikel.github.io.git`

2) Open `index.html` in your browser  
– On Windows: double‑click `index.html`  
– Or use a simple server: `python -m http.server 8000` and visit `http://localhost:8000/`

## Tech

- HTML, CSS, a small bit of vanilla JS (theme toggle and visitor counter)
- Served via GitHub Pages
